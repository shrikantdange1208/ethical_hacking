Check Response Headers - They tell u what kind of Protections are there
One of the Defense is HSTS - Http Strict Transport Security
Should be set in order to prevent an attack where we take secure data say https, we strip it, called Protocol Downgrade Attack (Man in the middle attack) and we make it http. This unencrypts the data.

Use securityheaders.com to analyze what protections are present for a website.

Look at level of encryption on a website:
nmap -script=ssl-enum-ciphers -p 443 tesla.com
